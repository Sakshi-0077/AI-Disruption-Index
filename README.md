# AI Disruption Index

## Overview

This project analyzes the impact of Artificial Intelligence on careers, industries, and workforce trends using data analysis and visualization techniques.

## Objectives

* Explore the influence of AI across different sectors.
* Identify industries most affected by AI adoption.
* Analyze workforce and career-related trends.
* Visualize key insights from the dataset.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Colab Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights and Conclusions

## Results

The analysis highlights how AI is transforming industries, creating new opportunities, and reshaping workforce demands.

## Repository Structure

* `AI-Disruption-Index.ipynb` - Main notebook containing analysis and visualizations.
* `README.md` - Project documentation.
